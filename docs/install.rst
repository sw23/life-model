============
Installation
============

Install the latest release from PyPI:

.. code-block:: bash

    pip install life-model

To work from a source checkout (editable install with development dependencies):

.. code-block:: bash

    git clone https://github.com/sw23/life-model.git
    cd life-model
    pip install -e . -r requirements-dev.txt

life-model requires Python 3.11 or newer.
